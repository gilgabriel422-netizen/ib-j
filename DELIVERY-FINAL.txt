CÓMO USAR:
─────────────────────────────────────────────────────────────────────────────

PARA CLIENTES:
  1. Login en http://localhost:3000
  2. Ir a: /enviar-atencion
  3. Enviar mensaje (Asunto + Contenido)
  4. Ver historial de respuestas

PARA EQUIPO DE ATENCIÓN:
  1. Login como atencion@crm.com
  2. Ir a: /dashboard-atencion
  3. Click: "📧 Bandeja de Mensajes"
  4. Responder mensajes
  5. Ver estados (Pendiente/En proceso/Respondido)

ENDPOINTS API:
─────────────────────────────────────────────────────────────────────────────

GET    /api/mensajes              → Obtiene todos los mensajes
POST   /api/mensajes              → Crea nuevo mensaje
PUT    /api/mensajes/:id          → Actualiza mensaje (respuesta)
DELETE /api/mensajes/:id          → Elimina mensaje

Respuesta esperada:
{
  id: 1,
  asunto: "Consulta",
  contenido: "Mi pregunta...",
  usuario_id: 5,
  estado: "pendiente|en_proceso|respondido",
  respuesta: "La respuesta...",
  fecha_creacion: "2024-01-15T10:30:00Z",
  fecha_respuesta: "2024-01-15T14:00:00Z"
}
